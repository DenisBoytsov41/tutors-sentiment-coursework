# Project overview

Проект курсовой работы по теме:
«Оценка настроения по текстам в ИС "Тьюторы" на основе дообученной модели анализа тональности».

Основной исследовательский цикл:
1. Подготовка среды
2. Понимание данных
3. Подготовка данных
4. Построение целевого датасета
5. Baseline
6. Дообучение и оценка
